Professional Background

Computer Security and Privacy Laboratory

Since June 2023, I have been working alongside Doctor Ning Zhang and PhD candidate Oliver Chang on several project regarding computer security in a healthcare setting.

  • Detecting Videos Harmful for Photosensitive Epilepsy Patients - For this project, I wrote a program using C++ and pthreads that will detect videos that are known to cause seizures or physical reactions according to the Web Content Accessibility Guidelines. The key criteria for detecting these videos were in regards to the frequency of flashes between frames and area of flash. For every video, I utilized OpenCV to process individual frames and stored the key information with a sliding window approach. Each frame was split into multiple sections processed through pthreads to allow for a faster runtime. This program had an average 34% decrease in runtime for a 40 second video and an average 67% decrease in runtime for a 5 second video compared to previous lab iterations.
  • Privacy in Women's Health Applications - For this project, I selected and systemized Women's Health Applications on Google Play Store. Next, I wrote scripts to conduct topic analysis on the play store reviews, static analysis on the permissions, libraries, and trackers on each application's APK file, and dynamic analysis with a UI monkey. The results of the analysis were compared to Google Play Store privacy guidelines and each application's privacy policy to identify discrepancies and necessary improvements to data privacy.
